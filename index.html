    Privacy & security

    GoblinPass Companion stores mappings only in this browser. It has no analytics, adverts, trackers, accounts, or network API.

    Mappings are encrypted with a non-exportable Web Crypto key stored by this browser. Device unlock uses WebAuthn where available. This local browser gate is not equivalent to Android Keystore protection and cannot protect a compromised device or browser profile.

    Encrypted backup files use PBKDF2-SHA-256 and AES-256-GCM. Losing the backup password means the file cannot be recovered.
